# Test Todo App

간단한 할 일 목록(Todo List)을 관리할 수 있는 웹 애플리케이션입니다.  
TypeScript + Webpack + ESLint + Prettier 환경에서 개발되었으며, GitHub Pages로 배포되었습니다.

## 배포 링크

👉 [https://wihyanghoon.github.io/test-todo](https://wihyanghoon.github.io/test-todo)

## 기술 스택

- TypeScript
- Webpack 5
- ESLint 9
- Prettier
- Babel
- Jest
- CSS Module Bundling
- GitHub Pages 배포
